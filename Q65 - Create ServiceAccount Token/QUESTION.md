# Question 65 – Create ServiceAccount Token

**Context:** You need to create a ServiceAccount and generate a token for it that can be used for API authentication.

## Your Task

1. Create a ServiceAccount named `api-service-account` in namespace `q65`.
2. Generate a token for the ServiceAccount and save it to the file `/tmp/sa-token.txt`.
